
# 双子星架构用户指南 (Dual Expert User Guide)

## 🌟 核心理念：红蓝军对抗 (Red Team / Blue Team)
为了提供更客观、反脆弱的决策支持，系统引入了 **双专家模型**：

1.  **🔵 蓝军 (Blue Team) - DeepSeek Reasoner (R1)**
    *   **角色**: 首席策略官 (Chief Strategist)
    *   **人设**: 激进、敏锐、寻找高赔率机会。奉行 LAG + GTO 交易哲学。
    *   **任务**: 全力生成“买入/卖出”的操作计划。

2.  **🔴 红军 (Red Team) - Google Gemini 2.0 Flash**
    *   **角色**: 首席风控官 (Chief Risk Officer)
    *   **人设**: 保守、怀疑、批判性思维。
    *   **任务**: 专门“找茬”。审查蓝军的计划，指出逻辑漏洞、潜在的流动性陷阱或宏观风险。

---

## 🚀 如何使用

### 1. 开启双专家模式
1.  在左侧栏设置 **DeepSeek API Key** 和 **Gemini API Key**。
2.  进入股票详情页 -> **复盘与预判**。
3.  点击 **"💡 生成复盘与预判"** 按钮。
4.  在弹出的配置面板中，勾选 **"🔴 开启红军审查 (Enable Red Team Audit)"**。

### 2. 解读双专家报告
你将看到左右互搏的精彩场面：
*   **DeepSeek 区域**: 展示买点、卖点、止损位与完整的思考链。
*   **Gemini 区域**: 展示 **风险评分 (Risk Score)** 和 **致命缺陷 (Critical Flaw)**。

### 3. 决策原则 (Decision Matrix)
| DeepSeek 建议 | Gemini 风险评分 | 决策建议 |
| :--- | :--- | :--- |
| **买入 (Buy)** | **低风险 (<4)** | ✅ **强力买入 (Strong Buy)** |
| **买入 (Buy)** | **中风险 (4-7)** | ⚠️ **谨慎跟进 (Cautious)** - 降低仓位 |
| **买入 (Buy)** | **高风险 (>7)** | 🛑 **拒绝执行 (Reject)** - 听从红军建议，观望 |
| **观望/卖出** | 任意 | ⏸️ **空仓观望 (Wait)** |

## ⚠️ 常见问题
**Q: 如果双专家意见冲突听谁的？**
A: **听风控官 (Gemini) 的**。正如德州扑克中 "Fold is cheaper than Call"，本系统的核心原则是生存优先。如果 Gemini 发现了 DeepSeek 忽略的风险，宁可错过机会，也不要冒进。
