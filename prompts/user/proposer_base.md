【盈亏状态-策略强制映射】(新增)
- 浮盈 > 20%: 必须考虑止盈(减仓/清仓)，禁止加仓、观望
- 浮盈 0-20%: 保本优先，观望或减仓，禁止加仓
- 浮亏 0-5%: 严格止损，反弹减仓，禁止加仓
- 浮亏 > 5%: 立即止损或反弹减仓，禁止观望、加仓

【反同质化检查】(新增)
查询该标的前3次策略建议:
- 如果3次都是"持仓观望": 本次必须给出不同判断(买入/卖出)
- 如果3次都是"减仓": 本次必须考虑是否止跌企稳
- 在"指挥官寄语"中说明: "前3次建议XX，本次改变因为..."
禁止连续给出相同建议超过3次。

【场景对策】(强制模块，缺失则策略无效)

必须给出3个场景，每个场景包含:
1. 触发条件: 具体价格区间 + 量能条件 + 时间窗口
2. 发生概率: A(>50%)、B(30-50%)、C(<30%)
3. 具体操作: 买入/卖出 + 具体股数 + 具体价格

示例格式:
场景A (强势反弹，概率60%):
  触发: 开盘 > 1.85 且 成交量 > 昨日50%
  操作: 持仓观望，突破1.90减仓10,000股@1.91

场景B (横盘震荡，概率30%):
  触发: 1.80-1.85 之间波动
  操作: 不做T，持仓观望

场景C (弱势下跌，概率10%):
  触发: 跌破1.80 或 10:30前跌幅>3%
  操作: 立即减仓20,000股@1.79止损
